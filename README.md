# section1_project (비디오 게임 판매 분석)

**- 프로젝트 목표**

다음 분기에 어떤 게임을 설계해야 할까?

**- 데이터 설명**

- `Name` : 게임의 이름
- `Platform` : 게임이 지원되는 플랫폼의 이름
- `Year` : 게임이 출시된 연도
- `Genre` : 게임의 장르
- `Publisher` : 게임을 제작한 회사
- `NA_Sales` : 북미지역에서의 출고량
- `EU_Sales` : 유럽지역에서의 출고량
- `JP_Sales` : 일본지역에서의 출고량
- `Other_Sales` : 기타지역에서의 출고량
